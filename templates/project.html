Project Information - Plant Disease Detection

This project focuses on detecting potato leaf diseases using deep learning techniques. Below are the steps undertaken in this project:

  • Data Collection: We collected a dataset of potato leaf images, including healthy leaves and leaves affected by early blight and late blight.
  • Data Preprocessing: The images were resized to 256x256 pixels and normalized to ensure consistent input to the model.
  • Data Augmentation: To enhance the dataset, data augmentation techniques like random flipping and rotation were applied to the images.
  • Model Creation:
    • Model Used: We employed a Convolutional Neural Network (CNN), which is particularly effective for image classification tasks.
    • Technologies: The model was built using TensorFlow and Keras, powerful libraries for deep learning and neural networks.
    • Architecture: The CNN consists of multiple convolutional layers to detect features, pooling layers to reduce dimensionality, and dense layers for the final classification.
  • Model Training: The model was trained on the preprocessed dataset with an 80-10-10 split for training, validation, and testing data, respectively. The training was performed over 25 epochs with a batch size of 32.
  • Model Evaluation: The model's performance was evaluated using accuracy and loss metrics on both the training and validation datasets. The final model achieved satisfactory accuracy and was saved for deployment.
  • Deployment:
    • The trained model was integrated into a Flask web application. Users can upload images of potato leaves, and the application will predict the disease and provide a confidence score.
    • The Flask application handles image uploads, pre-processes the images, and uses the trained model to make predictions.
  • Technologies and Libraries Used:
    • Programming Language: Python
    • Libraries: TensorFlow, Keras, Flask, NumPy, Pandas, Matplotlib
    • Environment: Google Colab for training the model, Visual Studio Code for running the Flask application
